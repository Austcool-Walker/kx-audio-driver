
"SA.minus"
A demo/reference kX DSP Plugin implementation of Spectrum Analyzer
(see http://www.driverheaven.net/effects-dsp/176505-spectrum-analyzer-possible-dsp.html for the origins and evolution).

-------------------------
Building:
* Add the "h" directory of the kX SDK to include directories
* Add the "lib" directory of the kX SDK to library directories
* Link to kxapi.lib and gdiplus.lib
* Compile as non-MFC and non-unicode dll
