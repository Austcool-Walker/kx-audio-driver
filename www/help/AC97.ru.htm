Кодек AC97

Окно AC97 kX микшера позволяет управлять входами и выходами кодека AC97 (обратите внимание, что все kX-совместимые платы имеют AC97 кодек, за исключением E-mu APS и Audigy2 Platinum eX).

Выходы:

TAD Выход
Выходной уровень разъёма, подключаемого к голосовому модему (моно). Обычно вход и выход подключаются через внутренний 4-штырьковый разъём, маркируемый "TAD".

PC Динамик
Выходной уровень PC Динамика (моно). Обычно подключается через внутренний 2-штырьковый разъём, маркируемый "PC_SPK".

Входы:

TAD Вход
Входной уровень разъёма, подключаемого к голосовому модему (моно). Обычно вход и выход подключаются через внутренний 4-штырьковый разъём, маркируемый "TAD".

Микрофон
Входной уровень иикрофонного входа (моно). Обычно исполнен в виде 1/8" стерео джека в задней части платы. Обратите внимание, что предпочтительней использовать линейный вход с внешними предусилителями, так как Микрофонный Вход имеет тенденцию к сильным шумам из-за дешевых встроенных компонентов усилителя.

Линейный вход
Входной уровень линейного входа (стерео). Обычно исполнен в виде 1/8" стерео джека в задней части платы.

CD
Уровень Аналогового входа CD (стерео). Обычно подключается через внутренний 4-штырьковый разъём, маркируемый "CD_IN" (не путать с цифровым входом CD_SPDIF).

Видео
Уровень видео входа (стерео). Обратите внимание, что на большинстве kX-совместимых плат этот вход не предусмотрен изготовителем (никакое аппаратное подключение не доступно).

Дополнительный
Уровень дополнительного входа (стерео). Обычно подключается через внутренний 4-штырьковый разъём, маркируемый "AUX_IN"

Другие функции:

Источник записи AC97
Источник записи AC97 выбирается через опускающееся меню, можно выбрать микрофон, CD, видео, дополнительный, линейный, стерео микс, моно микс или TAD. Обратите внимание, что одновременно можно записывать только с одного аналогового источника AC97, если только не выбрано СтереоМикс (или МоноМикс), позволяющее записывать вместе несколько AC97 источников с PCM (системный звук) выходом. Для дальнейших подробностей обратитесь к сигнальной блок-схеме.

Настройки AC97 кодека можно сбросить к настройкам по умолчанию нажатием кнопки AC97 Reset.

Опция увеличения чувствительности микрофона на 20dB включается щелчком по кнопке x2 Boost.
