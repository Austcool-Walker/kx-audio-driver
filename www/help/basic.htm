Basic Control Functions

kX Mixer uses single or double sliders to set levels, depending on the nature of the control item (mono or stereo).

Single (mono) slider

Double (stereo) slider

Slider levels can be set by clicking and dragging with the mouse, by using the page-up and page-down keys (for large increments) or by using the arrow-up and arrow-down keys (for small increments). The mouse-wheel can also be used for fine-tuning slider settings.

Double sliders may be linked so that they move together. Linked sliders display the Linked icon

To unlink, click the icon; unlinked sliders display the Unlinked icon

Most controls can be muted by clicking on the Mute icon

Muted controls display the Muted icon; click to un-mute

Numbers above show the amount of the slider currently set.
