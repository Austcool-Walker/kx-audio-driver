Connecteur Orange :

  • mode analogique : (5.1, Audigy) voie centrale (gauche), caisson de basses (droite)

  • mode digital :
            a. signal gauche: (1024, 5.1, Audigy): spdifo0 (avant digital); 
            b. signal droit:
                    i.  (1024): spdifo3 (digital arrière) ouspdifo1 [11] (mixage avant & arrière) [12]
                    ii. (5.1, Audigy); spdifo3 (arrière digital)

  • troisième signal: (5.1, Audigy): central digital/caisson de basses (spdifo2 [13])


 

 

Note: SPDIF#1 correspond à SPDIFO0; SPDIF#2 correspond à SPDIFO3 ; le troisième signal qui n'est pas sur la photo (il doit se situer avant 'GND' pour les cartes 5.1 et Audigy)

Limitations concernant les entrées/sorties : (selon les PDF officiels)

  • Les entrées Ligne 2/Micro 2 ne peuvent être utilisées simultanément avec l'I2S intégré.
  • L'entrée auxiliaire 2 ne peut être utilisée simultanément avec l'entrée coaxiale/optique SPDIF (sauf en mode contournement [bypass]).

Cartes filles :

Carte digitale E/S (originale, modèle CT4660):

  • Entrée MIDI
  • Sortie MIDI
  • Entrée coaxiale RCA
  • Sortie coaxiale RCA
  • DIN digitale

Carte optique digitale (modèle CT4770):

  • Entrée ligne 2 (*)
  • E/S digitale (pour le module digitale d'entrée/sortie)
  • DIN digitale
  • Entrée MIDI
  • Sortie MIDI  

(*)  uniquement disponible si connecté via la broche AUD_EXT

Module d'E/S digital (modèle CT4800): -- connecté à la carte fille "Digitale E/S

  • Entrée optique
  • Sortie optique
  • Entrée coaxiale
  • Sortie coaxiale

Module digital (modèle CT4801): -- connexion sur le 'jack orange'

  • Sortie coaxiale (spdifo 0) -- avant
  • Sortie optique (spdifo 0) -- avant
  • Sortie coaxiale (spdifo 2(?))  arrière
  • Sortie optique (spdifo 2(?))  arrière

LiveDrive (CT4860), LiveDrive II (CT4861), LiveDrive IR (SB0010), AudigyDrive (?):

  • Entrée coaxiale
  • Sortie coaxiale
  • Entrée optique (*)
  • Sortie optique (*)
  • Entrée auxiliaire n° 2 (*)
  • Casque
  • Entrée micro 2 / ligne 2
  • Entrée MIDI
  • Sortie MIDI
  • SB1394 (carte Audigy uniquement)
  • Infrarouge (LiveDrive IR, baie Audigy)

(*)  disponible sur les LiveDrive II et modèles ultérieurs
