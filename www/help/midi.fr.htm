Implémentation actuelle du pilote audio kX :

[outdated - see English version for details]
 0x80 Note relâchée // note off
 0x90 Note appuyée/relâchée // note on / off
 0xa0 Pression de touche -- changement de vélocité temps-réel // key pressure -- realtime velocity change
 0xb0 Contrôleur // controller
 0xc0 Changement de programme // program change
 0xd0 Pression canal (toutes les touches) -- changement de modulation temps-réel // channel pressure -- realtime modulation change
 0xe0 Modulation de hauteur via la molette // pitchbend
 0xf0, 0xf7 Système exclusif // sysex

 Contrôleurs :
  0 - Banque LSB (octet de poids faible) // bank LSB
  1 - Modulation de la hauteur // modulation
  6 - Entrée de données // data entry
  7 - Volume // volume
  8 - Balance
  11 - Expression // expression
  10 - Panoramique // pan = balance
  32 - Banque MSB (octet de poids fort) // bank MSB
  38 - Entrée de données // data entry
  64 - Maintien // sustain
  66 - Sostenuto // sostenuto
  67 - Pédale de sourdine -- filtre de coupure //soft pedal -- filter_cutoff
  69 - Tenue 2 -- modulation / volume enveloppe / maintien 2 /// hold2 -- mod/vol env release time
  91 - Réverbération // reverb
  93 - Effet de chœur // chorus
  96 - Augmentation des valeurs //data increment
  97 - Augmentation des valeurs // data decrement
  98 - LSB (octet poids faible) NPRN (Paramètres non enregistrés) fine // NRPN lsb fine
  99 - MSB (octet de poids fort) NPRN (Paramètres non enregistrés) basse résolution // NRPN msb coarse
  100 - LSB (octet de poids faible) RPN (Paramètres enregistrés) fine // RPN lsb fine
  101 - RPN (Paramètres enregistrés) MSB (octet de poids fort) basse résolution // RPN msb coarse
  120 - Tous les sons arrêtés [arrête toutes les notes] // all sounds off [terminate all notes]
  121 - Réinitialisation des contrôleurs // reset all controllers
  123 - Tous les sons arrêtés [arrête toutes les notes] //all notes off [release all notes]

 RPNs (Paramètres enregistrés) :
  00 - Sensitivité de la molette hauteur de son //pitchbend sensivity
  01 - Réglage fin général // master fine tuning
  02 - Réglage basse résolution général // master coarse tuning
  7f - Réinitialisation RPN // RPN reset

 Système Exclusif :
  f0 7f 7f 04 01 LL MM f7 - Volume synthétiseur // synth volume
