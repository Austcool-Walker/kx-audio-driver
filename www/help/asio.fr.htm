kX ASIO Routage et lecture [OUTDATED]

Généralités

La section lecture ASIO consiste en 16 bus mono situés sur le FXBus sur les sorties de 0 à 15 dans l'Écran DSP kX. Par défaut les bus sont partagés avec les autres flux sonores comme l'audio, le synthétiseur et les effets. Vous avez toujours la possibilité de créer votre propre routage ASIO, et de le configurer pour votre usage personnel. Cela peut comporter l'utilisation d'effets internes au DSP et/ou un mixage au niveau de la spatialisation.

Routeur kX ASIO

Les fonctions du routeur ASIO kX sont identiques à celle de l'Écran de routage kX.

Le routeur ASIO s'ouvre en sélection de bouton kX ASIO dans le Mixeur kX. Chaque bus mono dispose des ses paramètres de routage et peut être routé simultanément vers un total de quatre bus. (Une sortie stéréo dans votre hôte ASIO consiste en deux FXbus associés par une paire de nombres [paire/impaire]).

Un bus ne peut pas être routé vers le même bus plus d'une fois.

Le niveau du signal est réglé pour chaque flux en ajustant la valeur du curseur.

Dans l'exemple à gauche kxOUT00 est routé vers les bus 0, 15, 13 et 14, et seuls les bus 0 et 14 sont activés. Dans ce cas le bus 0 est le signal d'origine, pendant que le bus 14 est connecté aux chorus stéréo qui fonctionne comme un effet aller/retour.

Note: Cela dépend de vos paramètres DSP actuels.

Pour utiliser différents bus dans votre programme hôte ASIO vous devez :

  • Vérifiez que les sorties soient activées dans le(s) logiciel(s) ASIO hôte,
  • Sélectionnez la sortie correspondant (mono or stéréo) dans votre logiciel.

Panneau de contrôle ASIO

L'écran de configuration ASIO peut être ouvert à partir de votre programme hôte ASIO. Utilisez cette fenêtre pour ajuster la latence ASIO...

Le bouton ASIO kX ouvre la fenêtre routage ASIO.

 

Augmenter le temps de latence peut s'avérer utile si vous entendez des craquements dans vos enregistrements audio ou lors de l'utilisation d'un synthétiseur. La valeur minimale de latence pouvant être obtenue dépend de votre matériel, des ressources système et de la charge du système.

Le pilote ASIO supporte des temps de latence dans l'intervalle du 'presque temps réel' 2.66 ms à 341.33 ms.

En raison de limitation matérielle le pilote ASIO kX est figé sur la fréquence d'échantillonnage de 48kHz.

Trucs et astuces.

Cubase VST vous permet uniquement une sortie stéréo par défaut (la 'Master'). De nouvelles sorties peuvent être activées dans le mixeur Cubase. Il faut étendre la fenêtre pour faire apparaître un total de 8 sorties stéréo. Elles sont activées en pressant 'Activer' pour chaque bus que vous souhaitez utiliser. La paire de bus FX kX peut être sélectionnée sous chacun des curseurs. Les nouvelles sorties sont accessibles dans le mixeur VST pour chacun des canaux.

Logic Audio autorise automatiquement toutes les entrées/sorties ASIO quand ce type de pilote est sélectionné. Les entrées/sorties peuvent être sélectionnées à partir du mixeur Audio. Vous pouvez diminuer le nombre d'E/S ASIO dans l'Audio Hardware and Drivers setup de Logic.

Logic Audio Platinum configure automatiquement quatre haut-parleurs pour un mixage spatialisé à travers les sorties 1-4 (FXBus 0-3) - mais il peut aussi être configuré pour mixer un total de 9 haut-parleurs (8 canaux + l'effet basse fréquence).

Pour plus d'information concernant l'utilisation et la configuration ASIO, consultez la documentation spécifique de vos logiciels.
