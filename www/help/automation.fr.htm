Automation MIDI kX

Généralités

Quand un module d'extension est chargé dans la Fenêtre DSP un nombre lui est attribué et il apparaîtra dans la configuration de Automation MIDI kX. Les paramètres de contrôle disponibles apparaîtrons en haut à droite de la fenêtre quand un module d'extension sera sélectionné...

Vous pouvez ici associer les différents contrôles MIDI à un paramètre. Les contrôles MIDI disponibles sont : Contrôleur, Hauteur (Pitch Bend) et les messages Note appuyée/relâchée sur 16 canaux MIDI.

La plage des notes de messages de contrôles va de 0 à 127. La plage de hauteur (pitch bend) comporte 16384 étages ce qui permet un contrôle correct pour une gamme étendue et des plages de fréquence larges.

Les paramètres peuvent être contrôlés par n'importe quel programme MIDI à travers le contrôle des ports MIDI kX ou via le kXi VST instrument.
