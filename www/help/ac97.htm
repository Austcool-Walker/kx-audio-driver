AC97 Codec

The kX Mixer AC97 page allows control over the AC97 Inputs and Outputs (note that all kX-compatible cards have an AC97 codec, with the exception of the E-mu APS and Audigy2 Platinum eX).

Outputs:

TAD Out
Telephone Answering Device output level (mono). Typically accessed via an on-board 4-pin input and output connector labeled "TAD".

PC Speaker
PC Speaker output level (mono). Typically accessed via an on-board 2-pin connector labeled "PC_SPK".

Inputs:

TAD In
Telephone Answering Device input level (mono). Typically accessed via an on-board 4-pin input and output connector labeled "TAD".

Mic
Microphone input level (mono). Typically accessed through a 1/8" stereo jack at the rear of the card. Note that in general it is preferable to use the Line Input with an external preamp (pre-amplifier) if possible, since the Mic input tends to be noisy due to low-cost/performance built-in amplifier components.

Line
Line-In input level (stereo). Typically accessed through a 1/8" stereo jack at the rear of the card.

CD
CD Analog-In level (stereo). Typically accessed via an on-board 4-pin connector labeled "CD_IN" (not to be confused with the digital CD_SPDIF input).

Video
Video-In level (stereo). Note that on most kX-compatible cards, this input is not implemented by manufacturer (no hardware connection point available).

Aux
Auxiliary-In level (stereo). Typically accessed via an on-board 4-pin connector labeled "AUX_IN"

Other functions:

AC97 Recording Source
AC97 recording source is selected via the AC97 drop-down menu Choices are Mic, CD, Video, Aux, Line-In, Stereo Mix, Mono Mix or Phone. Note that only one analog AC97 source can be recorded at a time, unless StereoMix (or MonoMix) is selected, in which case multiple AC97 sources will be recorded together with PCM (system audio) output. Refer to the Signal Flow Diagram for further details.

The AC97 Codec may be reset to default settings by clicking on the AC97 Reset button.

Microphone 20dB Boost option is enabled by clicking on the x2 Boost button.
