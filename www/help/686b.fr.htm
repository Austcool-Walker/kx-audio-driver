Introduction :

En décembre 2000 VIA annonça son nouveau Southbridge 686B, en même temps que le chipset KT133A. Le nouveau Southbridge diffère de ses prédécesseurs par un support natif de l'ATA100.

Avec ce nouveau Southbridge et les cartes mères KT133A, différents utilisateurs ont rencontré des problèmes de corruption de données et d'instabilité système lors de transfert de fichiers, de l'écoute de musique, ou de toute action utilisant le bus PCI. D'une manière générale, les utilisateurs dotés de chipset KT133A, du SouthBridge 686B et de carte son Sound Blaster Live ont été affectés.

Nos tests montrent que ce problème n'est pas limité aux SBLive!, mais que les systèmes comportant des cartes Audigy peuvent aussi rencontrer des problèmes. Pire encore, avec une système affecté, le pilote kX est susceptible de ne pas être reconnu par le package des pilotes VIA 4in1, indiquant par là que le correctif VIA n'est peut être pas correctement installé.

Les correctifs fournis par les constructeurs de BIOS et par VIA ne mettent pas complètement fin à ce problème. Même après avoir corrigé les problèmes de corruption de données vous risquez toujours de rencontrer des craquements lors de la copie de fichiers, de la gravure de cédérom ou du transfert de données via le réseau.

Cependant nous avons trouvé une solution qui résout complètement ce problème. Cette solution nécessite de suivre les étapes suivantes :

  • Connectez vous sur le site du constructeur de votre carte mère et téléchargez le dernier BIOS de votre carte,
  • Connectez vous sur le site de VIA et téléchargez les derniers pilotes Via 4in1,
  • Installez le correctif (patch) 'George's PCI Latency' (c'est la seule solution pour résoudre ce problème dans certains cas).

Qui a besoin de ce correctif ?

Toute personne utilisant une carte mère basée sur le 686B, quel que soit le système d'exploitation utilisé.

Conclusion :

Nous vous conseillons d'utiliser le correctif 'George's PCI Latency' version 0.19 disponible ici*.
Vous devez également installer les derniers pilotes VIA disponibles ici et mettre à jour le BIOS de votre système.
