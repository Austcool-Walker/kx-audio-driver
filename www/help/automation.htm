kX Automation

Overview

When a plugin is loaded in the kX DSP window a number is assingned to it and it will appear in the kX Automation setup. Available control parameters will appear in the top right window when a plugin is selected.

Here you can assign different MIDI controls to a parameter. Available MIDI controls are Controllers, Pitch Bend and Note on/off messages on 16 MIDI channels.

The range of Note and Controller messages are from 0-127. The range of Pitch Bend is 16384 steps which makes it suitable for controlling large scales like frequency ranges.

The parameters are then controlable from any MIDI capable program through the kX Control MIDI port or via kXi VST instrument.
