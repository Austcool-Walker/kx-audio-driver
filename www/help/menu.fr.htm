Menu kX

Le Menu kX permet d'accéder aux fonctions du pilote kX, soit directement soit via les écrans de contrôle kX.

Le menu kX peut être affiché via un clic-droit sur l'icône   dans la barre système, (généralement situé en bas à droite de l'écran de l'ordinateur), ou en effectuant un clic-droit à l'intérieur de l'aire d'affichage du Mixeur kX ou de n'importe quel des écrans de contrôle.

Le menu kX est affiché de la manière suivante :

Menu Options:

Écrans de contrôle kX :

Mixeur kX

 Ouverture de l'écran principal du mixeur.

DSP kX

 Ouverture du kX DSP kX
Routeur kX Ouverture du Routage kX
ASIO kX   Ouverture de l'écran ASIO
Éditeur kX  Ouverture de l'éditeur

Périphériques         Ouverture du sous-menu de sélection du périphérique :

 

Pour les systèmes dotés de plusieurs cartes compatible-kX, ce menu permet de changer la carte actuellement sélectionnée.
Les changements effectués dans les écrans de contrôles kX affecteront uniquement la carte sélectionnée.

Configuration        Ouverture du sous-menu de configuration :

     

Sauvegarder la configuration
Sauvegarde la configuration de l'ensemble des écrans kX dans un fichier avec extension .kx ; le fichier par défaut est kxdefault.kx, vous pouvez cependant utiliser n'importe quelle dénomination.

Charger une configuration
Recharge une configuration préalablement sauvegardée pour tous les écrans kX. L'extension de fichier est .kx
Réinitialiser la configuration
Réinitialise tous les paramètres du pilote et du mixeur à leur valeur par défaut. Assurez vous d'avoir fait une sauvegarde de votre configuration actuelle avant de réinitialiser.
Configuration des tampons 



Configuration de la taille des tampons mémoire. Pour un usage standard vous n'avez pas à modifier ces valeurs, cette facilité est cependant offerte afin de vous permettre de modifier certains réglages :
  • Taille de la mémoire Tank (TRAM). Les valeurs vont de rien à 2048 Ko. Ajustez la taille de mémoire hôte disponible pour l'utilisation de certains effets DSP (la TRAM est généralement utilisée pour les délais de lignes). La valeur par défaut est de 256 Ko. Il convient de noter que ce paramètre n'influe pas sur la taille totale de mémoire disponible pour le microcode du DSP (valeur fixée au niveau matériel).
  • Réglage de la taille des tampons de lecture. La plage disponible va de 16384 octets (temps de latence estimé : 17.07) à 256 octets (latence : 0.26ms). Plus la taille est petite meilleur est le temps de latence, mais il est possible que ce ne soit pas utilisable en fonction de la configuration matérielle et logicielle utilisée.
  • Réglage de la taille du tampon mémoire d'enregistrement. Les valeurs autorisées s'échelonnent de 384 à 65536. La valeur par défaut est de 16384 octets. Normalement il n'est pas nécessaire de modifier ce paramètre.
  • Réglage du nombre de tampons AC3. La plage autorisée va de 2 à 8.
Générer un GUID 
Utilitaire de génération de GUID (Globally Unique IDentifier). Génère puis copie dans le presse-papier un GUID, prêt à être collé. Les GUIDs sont utilisés pour les sélecteurs d'apparence et également pour identifier des modules d'extension du DSP. Les utilisateurs standard n'ont pas besoin d'accéder à cette fonction.

Effets                 Ouverture du sous menu des modules d'extension :

Ce menu permet l'ajout de nouveaux objets DSP, il permet aussi l'enregistrement de modules d'extension en vue d'une utilisation ultérieure dans l'écran DSP kX.
Par ailleurs, les objets actuellement enregistrés ayant leur propres interfaces peuvent être accédés directement à partir de ce menu (pour ajuster leur paramètres).

Sélecteur d'apparence                    Ouverture du sous menu de sélection d'apparence :

Ce menu permet l'enregistrement de nouveaux modules d'apparence pour les utiliser dans le Mixeur kX ainsi que dans d'autres écrans de contrôle kX. L'extension par défaut pour les modules d'apparence est kxs (Remarque : ces fichiers sont en fait des fichiers ZIP, et peuvent être ouvert pour inspection ou modification en les renommant avec un suffixe .zip).
Par ailleurs, les modules d'apparence enregistrés peuvent être sélectionnés directement à partir de ce menu, ils seront affichés lors de la prochaine ouverture du mixeur kX (ou d'un autre écran).

Internet               Ouvre le sous menu Internet :

Ce menu permet d'accéder au site WEB du kxproject, il permet également de vérifier automatiquement si de nouveaux pilotes sont disponibles, il donne également accès à des effets DSP et à des modules d'apparence par des liens directs (remarque : cette fonctionnalité est en cours de développement).

Langage           Ouvre le sous-menu de sélection des langages.

Actuellement les langues supportées sont l'anglais et le russe. Remarque : le support de langage additionnel peut être effectué en éditant le fichier kxskin.ini. Ce fichier se trouve dans le fichier kxskin.kxs, qui est en fait un fichier ZIP (faire un clic-droit et sélectionner "ouvrir avec ..." puis sélectionnez winzip (ou un équivalent) pour l'ouvrir, vous pouvez également le renommer en .zip - n'oubliez pas de le renommer ensuite en .kxs quand vous avez terminé).

A propos ...                  Ouverture de l'écran d'information :

Déroulez la liste pour obtenir des informations détaillées, notamment la version des pilotes, du module d'apparence, le copyright, les informations sur la carte, les remerciements.
Tout rapport d'anomalie doit contenir l'ensemble du contenu de cette fenêtre. Copiez dans le presse-papier en effectuant un clic-droit dans le fenêtre de texte, puis choisir "sélectionner tout" et enfin "copier". Coller dans le rapport d'anomalie.

Quitter                     Fermer et quitter le mixeur kX.

Après avoir sélectionné Quitter dans le menu kX menu, le pilote et le mixeur kX resteront actifs, bien que le mixeur kX et les écrans associés soient fermés. Relancez kxmixer à nouveau ou réactivez le via un double-clic sur kxmixer.exe dans le répertoire windows/system32, ou bien à partir du menu démarrer de sélectionnez Exécuter, puis entrez kxmixer.exe dans la boite de texte enfin cliquez sur OK.
