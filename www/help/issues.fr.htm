Généralités

  • Par défaut les sorties haut-parleurs avant et arrière sont inversées (intentionnellement) - l'explication est ici.
  • EAX/EAX2.0/EAX3.0 ne sont pas complètement supportés.
  • Le contrôle de volume MIDI synthétiseur peut ne pas fonctionner en temps réel.
  • Si vous disposez d'un système compatible ACPI il est possible de vous deviez le désactiver l'ACPI. Voir le Guide ACPI pour plus de détails.

DSP

  • Positionner la taille de la TRAM à 0 (bien que non recommandé) peut causer des 'écran-bleus' Windows.
  • La sortie directe de flux AC3 (passthrough) n'est pas encore implémentée.

Fontes sonores (SoundFonts)

  • La taille maximale de l'échantillon des fontes sonores est susceptible d'être limitée par la configuration mémoire du système d'exploitation.

Contraintes matérielles

  • Les paramètres de phase, de niveau, etc... peuvent ne pas être optimisés pour certaines formes peu commune de version du codec AC97.

Contraintes liées aux systèmes d'exploitation

  • Un ajustement particulier du panneau de contrôle est nécessaire pour Windows 2000 (consultez le guide d'installation).
  • Les contrôles du mixeur Windows peuvent être doublés et un contrôle fantôme du cédérom audio est susceptible d'apparaître sous Windows 98 deuxième édition seulement.

Autres logiciels

  • Les utilisateur du Windows Media Player peuvent parfois rencontrer un comportement inattendu (par exemple saut de fichier ou passage en mode muet).
  • L'utilisation du module d'apparence tierce-partie "Lilac" peut causer des fuites de ressources (resource leakage).
  • Certains logiciels de visualisation de programme de télévision (comme ceux d'ATI) peuvent générer des conflits avec le pilote audio kX. Pour plus de détail consulter le site d'ATI.
  • Des problèmes de redémarrages intempestifs peuvent se produire. Pour plus de détails consultez le site de support de Microsoft.
