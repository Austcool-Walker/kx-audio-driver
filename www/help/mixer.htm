kX Mixer Overview

kX Mixer is an application which controls the kX Audio Driver. Access to Input and Output controls, and to the kX control applets, is provided through the kX Mixer user interface. Basic mixer control functions (common to most mixer applications) are explained here.

The icons found at the left of the mixer provide access to the different mixer control sets.
(Note that the following descriptions reference the default kX skin graphics. The default kX skin is "Aqua"). More on skins.
