Guide d'enregistrement ASIO [OUTDATED]

L'activation de l'enregistrement ASIO nécessite d'effectuer quelques modifications dans l'écran DSP kX. Cet écran s'affiche en sélectionnant le bouton DSP kX dans le mixeur, ou via un clic droit dans l'icône système puis en sélectionnant 'DSP kX DSP' depuis le Menu kX.

Les entrées ASIO se trouvent dans l'objet epilog (voir le Guide kX DSP pour des informations complémentaires sur ce sujet ainsi que sur les objets de base). Les 16 dernières entrées de cet objet correspondent aux entrées ASIO. Celle-ci permettent d'enregistrer simultanément 7 entrées stéréo ou 16 entrée mono (ou n'importe quelle combinaison de celles-ci).

Par défaut aucune source n'est connectée sur ces entrées, vous devrez donc modifier la source que vous voulez enregistrer en ASIO.

Les entrées de la carte son sont situées dans l'objet prolog. Les sorties sur le 'prolog' représentent les entrées physiques de votre carte son (ainsi que des baies frontales, cartes filles et périphériques externes connectés sur la carte). Ces connecteurs sont des paires stéréo et correspondent aux curseurs du bas situés dans la fenêtre Entrées & sorties du Mixeur kX.

Sur une carte 10k1 le prolog représente les signaux d'entrées suivants (7 paires stéréo):

In0/1: Entrée AC97
In2/3: Entrée cédérom digital
In4/5: (? 1)
In6/7: Entrée coaxiale / Entrée optique (LiveDrive/Carte fille)
In8/9: Ligne 2/micro 2 in (LiveDrive)
In10/11: (? 2)
In12/13: Entrée auxiliaire n° 2 (LiveDrive)

Exemple - Enregistrement via l'entrée ligne

Les entrées analogiques sont situées directement sur la carte son (Ligne/Micro/Auxiliaire/Cédérom/TAD/Vidéo) peuvent être sélectionnées à partir de la fenêtre AC97 du Mixeur kX, elles peuvent être enregistrées soit seules soit en mixage stéréo ou mono (mais pas avec toutes les combinaisons - consultez le Diagramme de flux des signaux kX pour plus d'information). Pour enregistrer l'entrée ligne, à partir de la fenêtre AC97 du mixeur kX sélectionnez Entrée ligne, laissez tous les curseurs muets et aller à la fenêtre Entrées & sorties du mixeur. Vous pourrez ici utiliser le curseur AC97 pour ajuster le niveau. Dans l'écran DSP kX vous devrez connecter prolog in0 et in1 avec epilog asio0 etasio1 comme indiqué ci dessous (consultez le Guide DSP kX pour les détails sur la mise en place de connexions) :

Le pilote ASIO est prêt pour enregistrer une entrée AC97. Remarquez que le connecteur par défaut sur le prolog n'a pas besoin d'être supprimé.

Maintenant vous devez activer les entrées ASIO dans votre application d'enregistrement ASIO préférée (comme Cubase, Logic, etc.). Il peut être utile de désactiver le monitoring logiciel dans le logiciel pour éviter d'entendre deux signaux. Si vous enregistrez à partir d'une autre source simultanément, connectez simplement le signal sortie du prolog vers une autre paire ASIO et activez cette entrée dans votre programme hôte.

Autres astuces

Vous pouvez virtuellement connecter n'importe quel connecteur de sortie dans l'écran kX DSP aux entrées ASIO et enregistrer ce qui est présent en sortie. Cela signifie que vous pouvez enregistrer des signaux de type réverbération, mixage final ou canal de synthétiseur.

Astuces ASIO

Dans Cubase, vous devez activer l'option jouer en arrière plan pour permettre à Cubase de continuer à émettre des sons quand vous effectuez une lecture dans d'autres logiciels (ou ajuster les paramètres du DSP).
