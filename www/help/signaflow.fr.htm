Digramme des flux Entrées/Sorties / kX

En mode analogique (défaut pour les cartes sans processeur 10k2) :

Le volume PCM est constant (=0 – pas d'atténuation) ; Volume général idem (=0) ; Spatialisation idem (=0) ; il est possible d'enregistrer/monitorer une source analogique via la 'Source d'enregistrement' ; tous les volumes d'entrée AC97 sont mis à 'muet' ; si ils sont activés ils sont routés directement vers les haut-parleurs avant. L'utilisation du mode 'Mixage stéréo' ou l'activation de plusieurs sorties AC97 n'est pas recommandée. Le volume est contrôlé via le DSP entrée AC97 (in0/in1).

En mode 'Sortie digitale uniquement :

Les volumes PCM AC97, Spatialisation et le volume général sont 'muet' ; 'La source d'enregistrement' est 'Mixage stéréo'; cela permet à de multiples sources analogiques d'être simultanément enregistrées/monitorées (avec ou sans PCM – en fonction de la configuration du DSP kX); les entrées peuvent être contrôlées séparément. Il faut activer la source analogique AC97 désirée. Le volume est contrôlé par le volume AC97 source et l'entrée AC97 du DSP (in0/in1).

Le 10k2 a la possibilité d'enregistrer simultanément plusieurs sources analogiques quand le volume PCM est positionné à 'muet' et les sorties avant sont dirigées vers le codec I2S directement ; la sortie AC97 est en mode 'muet' par défaut ; les volumes d'entrées peuvent être contrôlés indépendamment ; le mode 'Sortie digitale uniquement' n'influe pas le codec AC97 ; il n'y a pas de sortie spatialisée AC97 – I2S est utilisé à la place.
