kX Command Line Reference

kxsetup.exe options Performs standard kX Audio driver and kX Mixer setup 
--clean
 Performs pre-install cleanup procedure which is done automatically when the kX drivers are installed. Disables some Creative software components and any previously installed instance of the kX Mixer and the kX Audio Driver 
--noreboot
 Cancels reboot after driver installation
kxmixer.exe options: 
--quit
 Closes all active instances of the kX mixer 
--once
 Quit kX Mixer after restoring the settings (should be used with '--startup') 
--startup
Performs startup sequence: restores kX settings from the registry 
--shell
Shell integration option: for internal use only 
--no10k2
Disables 10k2 support 
--10k2
Forces 10k2 support 
--no51
Disables 5.1 support 
--51
Forces 5.1 support 
--aps
Forces E-mu APS support 
--noaps
Disables E-mu APS support 
--noa2
Disables Audigy2 support 
--a2
Forces Audigy2 support 
--nok2ex
Disables Audigy2 Platinum Ex support 
--k2ex
Forces Audigy2 Platinum Ex support 
--no2k
Disables Windows 2000/XP Extended GUI support 
--ac97
Forces AC97 codec support 
--noac97
Disables AC97 codec support
