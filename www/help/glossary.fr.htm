Glossaire

AC97 Audio Codec 97 (Codec provient de Codeur-Décodeur). Couramment utilisé pour faire référence à l'ADC/DAC qui se trouve sur les cartes SoundBlaster Live! et Audigy.
ADC Analog to Digital Converter. Convertisseur analogique/digital. Un périphérique chargé de convertir les signaux analogiques dans leurs formes digitales.
APS Audio Production Studio. Une carte son basée sur l'Emu10k1 fabriquée par E-mu Systems Inc.
ASIO  Audio Stream Input Output. Flux audio d'entrées/sorties. Un protocole de transfert muti-canaux développé par Steinberg.
DAC Digital to Analog Converter. Convertisseur digital/analogique. Un périphérique chargé de convertir les signaux digitaux dans leurs formes analogiques.
DOO Digital Output Only. Sortie digitale uniquement : configuration du Codec AC97 permettant l'enregistrement simultané de plusieurs sources analogiques.
DSP Digital Signal Processing. Processeur de signaux digital.
E-Drive Module d'entrée/sortie APS E-mu APS destiné à pouvoir être installé dans les baies disques standard de type 5-1/4".
BusFx Bus d'effet. Un chemin partagé permettant le routage des différents signaux d'effets à partir d'une ou plusieurs sources vers une ou plusieurs destinations.
GPR General Purpose Register. Registre à usage général.
I2S    Inter-IC Sound, un format de son digital.
LiveDrive Module d'entrées/sorties destiné aux baies disques standards 5-1/4", se connecte sur les cartes Live! et Audigy.
MIDI    Musical Instrument Digital Interface. Interface digitale pour les instruments de musique. Protocole d'interconnexion et de contrôle des instruments de musique.
PCM Pulse Code Modulation. (Modulation par impulsions codées) Encodage non compressé pour le stockage de données digitales, généralement associé aux données audio de type Wave.

SPDIF    Sony-Philips Digital Interface Format, un format digital standard.
TAD     Telephone Answering Device (ou modem voix). Périphérique de réponse téléphonique . Connexion d'un modem à la carte pour messagerie vocale ...
