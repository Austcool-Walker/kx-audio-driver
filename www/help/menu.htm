kX Menu

The kX Menu provides access to the kX driver functions, both directly and through the kX Control Applets.

The kX Menu can be displayed by right-clicking the kX system tray icon, , which is found in the Windows System Tray (typically found at the lower right corner of the desktop), or by right-clicking within the display area of the kX Mixer or any of the kX control applets.

The kX Menu will appear as follows:

Menu Options:

kX Control Applets:

kX Mixer

 Opens the main kX Mixer applet.

kX DSP

 Opens the kX DSP control applet.
kX Router Opens the kX Router applet.
kX Editor Opens the kX Editor applet.
kX Automation Opens the kX Automation control applet.
kX Remote Opens the kX Remote control applet.
kX SoundFont Opens the kX SoundFont control applet.

Device        Opens the device selection sub-menu:

For systems with multiple kX-compatible cards, this menu allows the user to change the currently selected card.
Changes made in any kX control applet will affect currently selected card only.

Settings      Opens the Settings sub-menu:

Save Settings
Save settings for all kX applets as a file with the extension .kx. Default filename is kxdefault.kx, however, any filename may be used.

Load Settings
Load previously saved kX settings for all kX applets. The file extension for kX settings is .kx

Reset Settings
Resets all the kX Audio Driver and kX Mixer settings to their default values. Be sure to save your current settings before resetting.

Setup Buffers



Setup the kX buffers. In normal usage no adjustments will be required to the default buffer setup, however, facility is provided for adjusting the following settings:

  • Set Tank Memory (TRAM) Size. Range is from None to 2048kB. Adjusts the amount of host memory available for use by certain DSP effects (TRAM is typically used for delay lines). Default setting is 256kB. Note that this setting does not affect the total amount of memory available for DSP microcode (value fixed in hardware).
  • Set Playback Buffer Size. Ranges from 16384 bytes (estimated 17.07ms latency) to 256 bytes (estimated latency 0.26 ms). Smaller is faster, but may not be usable depending on system hardware and software configuration.
  • Set Recording Buffer Size. Range is from 384 to 65536. Default value is 16384 bytes. Normally no adjustment to this setting is required.
  • Set Number of AC3 Buffers. Range is from 2 to 8.

Generate GUID
GUID (Globally Unique IDentifier) generation utility. Generates and copies a GUID to the clipboard, ready to be pasted. GUIDs are used in skin editing and also to identify DSP plug-ins. Typical users will not need to access this function.

Effects         Opens the Effects Plug-ins sub-menu:

This menu allows new DSP objects and plugins to be registered for use in the kX DSP applet.
Also, currently registered DSP objects having User Interfaces can be accessed directly from this menu (for tweaking effects settings).

Skin             Open the Skins sub-menu:

This menu allows new skins to be registered for use with the kX Mixer and other kX control applets. Default extension for kX skins is kxs (note that kxs files are in fact zip or rar files, and can be opened for inspection or modification by renaming to zip or rar, depending on file).
Also, currently registered skins can be selected from this menu and will be displayed the next time the kX Mixer (or other kX applet) is opened.

Internet       Opens the Internet link sub-menu:

This menu provides a link to the kxproject website, and allows automatic checking for driver updates, as well as facilitating DSP effect and skin downloads via direct links (note that this feature is under development).

Language      Opens the language selection sub-menu.

Currently supported languages are English, Russian, Deutsch, Polish, Espanol, Romanian, French, Italian, Swedish, Norwegian, Dutch, Brasilian, Bulgarian, Croatian and Hungarian. Note that additional language support can be added by editing the file kxskin.ini. This file can be found within the file kxskin.kxs, which is in a fact a zip file (right-click and select "open with...", then choose winzip (or equivalent) to open, or simply rename to .zip - and don't forget to rename back to .kxs when you're done).

About...         Opens the kX Information applet:

Scroll down for a detailed information listing,including kX Control Version Number, Skin Info, Copyright, PCI Card Info and Credits.
Any bug reports should include the entire contents of this window. Copy to the clipboard by right-clicking within the text window, then choosing "select all" and then "copy". Paste into bug report.

Help               Opens this Help window.

Speaker Test  Opens the kX Speaker Test applet

Quit                Closes and Quits the kX Mixer.

After selecting Quit from the kX menu, both the kX Driver and the Windows mixer will still remain active, although the kX mixer and all kX applets will be closed. Run kxmixer again to re-activate - double-click  kxmixer.exe in the windows/system32 directory, or from the Windows Start menu select Run, then type kxmixer.exe in the text box and click OK. Newer versions also can be run from a folder 'kX Audio Driver' in Start Menu\Programs.
