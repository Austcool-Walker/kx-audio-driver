kX меню

kX меню обеспечивает доступ к функциям kX драйвера так же, как и окна управления.

kX Меню вызывается кликом правой кнопкой мыши на значке , который находится в системном лотке (system tray) Windows (обычно располагается в правом нижнем углу рабочего стола), или, щелкая правой кнопкой мыши в пределах области kX микшера или любого другого окна управления kX (кроме kX DSP).

kX меню появится в следующем виде:

Параметры меню:

Окна управления kX драйвером:

kX микшер

 Открывает главное окно kX микшера.

kX DSP

 Открывает окно управления kX DSP.
kX Router Открывает окно kX Router.
kX редактор Открывает окно kX редактора.
kX автоматизация Открывает окно управления kX автоматизации.
kX пульт Открывает kX пульт.
kX SoundFont Открывает окно управления kX SoundFont.

Устройства        Открывает подменю выбора устройства:

 

Для систем с несколькими kX-совместимыми платами это меню позволяет пользователю изменить выбранную в настоящее время плату.
Изменения, сделанные в любом окне управления kX, затронут только выбранную в настоящее время плату.

Настройки      Открывает подменю настроек:

Сохранить настройки
Позволяет сохранить настройки для всех окон kX как файл с расширением .kx. Заданное по умолчанию имя файла - kxdefault.kx, однако, можно использовать любое имя файла.

Загрузить настройки
Позволяет загрузить предварительно сохраненные настройки для всех окон kX. Расширение файла для kX настроек - .kx

Сбросить настройки
Сбрасывает все настройки kX аудио драйвера и kX Микшера в значение "по умолчанию". Убедитесь, что сохранили текущие параметры настройки перед сбросом..

Настройка буферов



Установка буферов. В нормальном состоянии не требует дополнительных корректировок заданных по умолчанию величин, однако, обеспечивается корректировка следующих величин:

  • Размер основной памяти (TRAM). Диапазон от 0 до 2048Кб. Задает количество основной памяти, доступной для использования некоторыми DSP эффектами (TRAM обычно используется для линий задержки). Параметр по умолчанию - 256Кб. Обратите внимание, что установка этого параметра не затрагивает общее количество памяти, доступной для микрокоманд DSP (значение, установленное аппаратными средствами).
  • Размер буфера воспроизведения. Диапазон от 16384 байт (с задержкой 17.07ms) до 256 байт (с задержкой 0.26 ms). Чем меньше величина, тем быстрее, но в этом случае она может быть не пригодна для использования, в зависимости от вашей аппаратной и программной конфигурации.
  • Размер буфера записи. Диапазон от 384 до 65536. По умолчанию величина равна 16384 байтам. Обычно не требуется никакой корректировки этого параметра.
  • Количество AC3 буферов. Диапазон от 2 до 8.

Сгенерировать GUID
GUID (Глобальный Уникальный Идентификатор) утилита генерирования. Генерирует и копирует GUID в буфер обмена, готовый для вставки. GUID используется при редактировании скинов и идентификации DSP плагинов. Рядовые пользователи не нуждаются в услугах этой функции.

Эффекты         Открывает подменю эффектов и плагинов:

Это меню позволяет подключить новые DSP объекты и плагины, и зарегистрировать их для использования в окне kX DSP.
Также, к текущим зарегистрированным объектам DSP, имеющим пользовательский интерфейс, можно обращаться непосредственно из этого меню (для редактирования параметров эффектов).

Скин             Открывает подменю скинов:

Это меню позволяет зарегистрировать новые скины для использования их в kX микшере и других окнах управления kX. Расширения файлов по умолчанию для kX скинов - kxs (заметьте, что файлы kxs - это фактически zip / rar архивы, которые вы можете открыть для просмотра, модификации или переименования).
Также, в этом меню вы можете выбрать уже зарегистрированные скины, которые вступят в силу при следующем открытии kX Микшера (или другого kX окна).

Интернет       Открывает подменю Интернет-ссылок:

Это меню обеспечивает связь с сайтом kxproject, позволяет автоматически проверять обновления драйвера и облегчить загрузку DSP эффектов и скинов через прямое соединение (обратите внимание, что эта возможность находится еще в разработке).

Язык      В настоящее время поддерживаются следующие языки: английский, русский, немецкий, польский, испанский, румынский, французский, итальянский, шведский, норвежский, голландский, бразильский, болгарский, хорватский и венгерский. Обратите внимание, что для дополнительной поддержки нового языка нужно просто отредактировать файл kxskin.ini. Этот файл вы можете найти в архиве kxskin.kxs, который фактически является zip архивом (щелкните правой кнопкой мыши и выберите "открыть с помощью", затем выберите winzip (или альтернативный архиватор) чтобы распаковать его, или просто переименуйте файл с расширением .zip - и не забудьте переименовать потом назад, в расширение .kxs когда закончите).

О программе...         Открывает информационное окно:

Прокрутите вниз для получения детальной информации, включающей номер версии kX драйвера, информацию о скинах, авторские права, информацию о PCI плате и благодарности.
Любые сообщения об ошибках должны включать содержимое этого окна. Для копирования в буфер обмена - щелкните правой кнопкой мыши в пределах этого окна, выберите "Выбрать все" и затем "Копировать". Вставьте в сообщение об ошибках.

Помощь               Открывает это окно помощи.

Проверка колонок  Открывает окно проверки колонок

Выход                Закрывает и выгружает kX микшер.

После выбора Выход из kX Меню, все kX драйвера и Микшер Windows будут оставаться активными, хотя kX Микшер и все остальные окна kX будут закрыты. Чтобы открыть Микшер снова, сделайте двойной щелчок на файле  kxmixer.exe, находящийся в директории windows/system32, или из меню Windows Пуск - Программы - kx Audio Driver - kx Mixer, или выберите меню Пуск - Выполнить, напечатайте kxmixer.exe и нажмите ОК.
