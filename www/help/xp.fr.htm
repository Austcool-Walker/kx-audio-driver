Installation sous Windows 2000/XP (versions francisées de Windows)

Ce document est basé sur des messages et des courriers électroniques de Trypt0. Il est conseillé d'essayer d'utiliser le kxsetup avant de tenter une installation manuelle, en effet les dernières versions de kxsetup devraient fonctionner sans aucun problème.

Pour installer le pilote manuellement sous Windows XP ou sous Windows 2000, commencez par télécharger les dernières versions du pilote audio kX ainsi que les bibliothèques MFC (en version debug).

N'oubliez pas de désinstaller LiveWare et/ou les pilotes APS. Vous pouvez conserver PlayCenter ou les autres logiciels Creative.

Ensuite, décompressez (unRAR) le pilote dans un répertoire temporaire, décompressez les bibliothèques MFC vers le répertoire \windows\system32 (ou \winnt\system32).

Après cette extraction, passez en mode 'Sans Echec' après avoir redémarrer l'ordinateur et en appuyant rapidement sur la touche F8 dès la fin de l'initialisation du BIOS (après les tests POST).

Vous devez maintenant vous trouver sous Windows en mode 'sans échec'.

Appuyez maintenant simultanément sur le bouton 'Windows' et sur la touche Pause ; cliquez sur le choix matériel ; cliquez sur Gestionnaire de périphériques ; double-cliquez sur "Contrôleurs son, vidéo et jeu" ; double cliquez sur le périphérique correspondant à la carte son ; cliquez sur pilote ; cliquez sur "Mettre à jour le pilote" ; cliquez sur "spécifier l'emplacement du pilote" ; cliquez sur disquette fournie et rechercher le pilote sur votre ordinateur (dans le répertoire temporaire) ; sélectionnez le pilote correspondant au modèle de votre carte.

Après l'installation vous devrez redémarrer l'ordinateur. Servez vous de l'utilitaire kX dans la barre système de Windows pour configurer votre carte son. (Configuration -> Réinitialisation de la configuration).

Vous pouvez également essayer de supprimer ou de renommer les fichiers \winnt\Driver Cache\i386\driver.cab ainsi que \winnt\inf\wdma10k1.inf et \winnt\inf\wdm_emu.inf avant de lancer kXSetup.
