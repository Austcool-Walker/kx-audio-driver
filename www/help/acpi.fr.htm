Si vous envisagez de désactiver l'ACPI, prenez en compte les remarques suivantes :
  • La désactivation de l'ACPI sur une installation existante peut entraîner un blocage votre système;
  • Pensez donc à protéger vos données sensibles, prenez une image disque ou une sauvegarde 'ghost' de vos applications;
  • Les améliorations obtenues pour l'audio ne seront peut être pas aussi importantes que celles attendues;
  • Les problèmes liés à l'ACPI se produisent généralement si votre carte son partage une IRQ avec une autre carte PCI ou AGP;
  • Avant de désactiver l'ACPI, essayez un autre emplacement pour votre carte pour voir si cela améliore la situation.
