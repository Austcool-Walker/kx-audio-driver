Les fontes sonores (SoundFonts)

Le mixeur kX permet de charger des fontes sonores en vue de leur utilisation par des synthétiseurs MIDI.

Pour charger une fonte sonore cliquez sur le bouton dans l'écran fontes sonores. Une boite de dialogue apparaîtra où vous pourrez sélectionner le fichier de fontes à charger (extensions de type .SF2).

Les fontes sonores actuellement chargées sont listées dans le panneau supérieur. La sélection d'une fonte dans ce panneau fait apparaître les informations la concernant dans le panneau inférieur, notamment : nom, moteur, copyright, ingénieur et statistiques.

L'utilisation mémoire est affichée dans la barre Memory Usage en bas de l'écran.

Pour décharger une fonte, sélectionner la dans le panneau supérieur puis cliquer sur l'icône :.

Le pilote kX supporte le chargement de fontes de types 2.0 et 2.1 uniquement, les fontes de type 1.0 (format .SBK) ne sont pas supportées. Il n'y a pas de support des propriétés de modulation des Sf 21.

Le pilote WDM kX affecte les fontes en mémoire cache à la demande ; il n'y a pas de contrôle du cache pouvant être ajusté par l'utilisateur. Théoriquement la taille maximale des fontes pouvant être chargées est limitée uniquement par les ressources système, cependant ce maximum varie entre 30 et 70% de la mémoire physique RAM, en raison de limitations dues à la gestion mémoire du noyau ainsi qu'à d'autres facteurs. Dans tous les cas seul 32 Mo d'échantillons peuvent être utilisés simultanément en raison de limitations matérielles.
