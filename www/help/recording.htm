Recording level controls

kX Audio Drivers support 'Direct SPDIF Recording'. Read Direct Recording guide for details.

The kX Mixer Recording control page provides access to the following record level controls:

Software Sources:

Master Recording Level
This stereo pair controls the overall recording level for all sources.

Wave Recording Level
This mono slider controls the recording level for the PCM Wave source.

Synth Recording Level
This mono slider controls the recording level for the MIDI synth source.

FX1 (Reverb) Recording Level
This mono slider controls the amount of FX1 (Reverb by default) applied to playback sources while recording.

FX2 (Chorus) Recording Level
This mono slider controls the amount of FX2 (Chorus by default) applied to playback sources while recording.

Hardware Sources:

AC97 Recording Level
This stereo pair controls the recording level for the AC97 source(s)

CD Digital Recording Level
This stereo pair controls the recording level for the CD Digital (CD_SPDIF) source.

'Unknown 1' Recording Level
This stereo pair controls the recording level for the Unknown 1 source.

'Unknown 2' Recording Level
This stereo pair controls the recording level for the Unknown 2 source.

Mic-2 / Line-2 Recording Level
This stereo pair controls the recording level for the Mic-2 / Line-2 source (for systems with LiveDrive only).

Coaxial Recording Level
This stereo pair controls the recording level for the Coaxial source (for systems with LiveDrive only).

Aux2 Recording Level
This stereo pair controls the recording level for the Aux2 source (for systems with LiveDrive only).

Analog-1/Analog-2 Recording Level (APS card only)
This stereo pair controls the recording level for the analog A1/A2 sources.

Analog-3/Analog-4 Recording Level (APS with Edrive only)
This stereo pair controls the recording level for the analog A3/A4 sources.

Digital D1 Recording Level (APS card only)
This stereo pair controls the recording level for the digital D1 source.

Digital D2 Recording Level (APS with Edrive only)
This stereo pair controls the recording level for the digital D2 source.
