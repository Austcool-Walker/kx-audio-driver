micromastering- plugins v1.0
----------------------------


microwidth v1.0-   enhances the stereo width from mono to more stereo. 


microbuttom v1.0-  simulates an "aphex big bottom" (bass enhancer) with 
                   12 db lowpass filter and antiparallel diode limiter. 


microexiter v1.0-  simulates an "aphex aural exciter typ b" (treble enhancer)
                   with 12 db highpass filter and antiparallel diode limiter/overdrive. 


microlimiter v1.0- limits the signal over 10 different transfer- curves (logaritmic).


put the plugins in this order for best results.
